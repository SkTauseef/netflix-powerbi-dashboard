# Netflix Power BI Dashboard

## Project Overview
Interactive dashboard analyzing Netflix movies and TV shows.

## KPIs
- Total Titles
- Total Directors
- Total Countries
- Start Date
- End Date

## Visualizations
- Movies vs TV Shows
- Titles by Genre
- Content Rating Distribution
- Geographic Distribution
- Yearly Trend Analysis

## Tools Used
- Power BI
- DAX
- Power Query

## Dataset
Netflix Titles Dataset
